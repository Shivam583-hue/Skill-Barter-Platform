# **Opportune Hub**

Opportune Hub is a multi-purpose collaboration platform where users can connect, create opportunities, and work together effectively. The platform includes features for real-time collaboration, DSA discussions, job opportunities, and showcasing portfolios.

---

## **🌟 Key Features**
1. **Collaboration & Chatrooms:**  
   - Create a new chatroom from the _Chatrooms_ section.  
   - Use the provided chatroom ID to create an opportunity in the _Post_ section.  
   - View comments on your opportunity post, check out profiles, and invite collaborators to your chatroom.  

2. **Direct Collaboration Requests:**  
   - Send project proposals directly from a user's profile.  
   - Manage all your proposals in the _Inbox_ page.  

3. **DSA Discussions:**  
   - Share and explore DSA-related content in the _DSA_ section.  

4. **Jobs & Referrals:**  
   - Discover job opportunities and referrals in the _Jobs_ section.  

5. **Flex Your Skills:**  
   - Showcase your projects, designs, or portfolios in the _Flex_ section to attract employers, clients, or collaborators.

## **💻 Tech Stack**
- **Language & Runtime:** TypeScript, Node.js  
- **Backend:** Express (REST APIs)  
- **Frontend:** React  
- **Database & ORM:** PostgreSQL (NeonDB), Prisma  
- **CSS Framework:** Tailwind CSS  
- **UI Libraries:** DaisyUI, Material UI (MUI), shadcn  
- **WebSockets Framework:** Socket.IO  
- **Deployment:** Render _(Free Tier)_
